Vi arbejder med use cases på følgende måde:
1. Vi laver en liste af use cases ud fra vores domæne model og ordanalyse (PO udsagn).
2. Udvider dem alle til brief.
3. Vi rangerer dem ud fra risiko og værdiskabelse.
4. Udvælger de vigtigste 10-20%, som laves til fully dressed.
5. De fully dressed use cases oversættes til tekniske implamenterings opgaver.
6. Herefter pointgives hver opgave i backloggen(ovenstående opgaver)
7. Vi laver en detaljeret plan over næste iteration og en general plan over alle iterationerne.
