| Use Case Name                       | Book transportmiddel                                                                                                                                                                                                                                                                                                                                                    |
| Primary Actor                       | Specialpædagog                                                                                                                                                                                                                                                                                                                                                          |
| Stakeholders and Interests          | **Pædagogerne:** vil gerne have en smertefri oplevelse med at koordinere brugen af transportmidler.<br>**Bostedet:** interesse i at have et godt arbejdsmiljø, hvilket kan påvirkes af hvor nem arbejdsgangen er for deres medarbejdere.<br>**Borgere:** interesseret i at deres støtte kommer til den aftalte tid, og at pædagogerne er fleksible i deres besøgstider. |
| Preconditions                       | Valgt gyldig tidsrum. (Se [[Se ledige transportmidler for et bestemt tidspunkt\|UC2]])                                                                                                                                                                                                                                                                                  |
| Success Guarantee                   | Transportmidlet er ikke længere tilgængelig i det valgte tidsrum og det står som booket i medarbejderens navn.                                                                                                                                                                                                                                                          |
| Main Success Scenario               | **1:** Ser ledige transportmidler<br>**2:** Vælger ledig transportmiddel<br>**3:** Booker valgte transportmiddel<br>**4:** Modtager bekræftigelse                                                                                                                                                                                                                       |
| Extensions                          | **A:** Vælger transportmiddel der allerede er booket.<br>**1:** Ser ledige transportmidler<br>**2:** Vælger ledig transportmiddel<br>**3:** Prøver at booker valgte transportmiddel<br>**4:** Modtager en fejlmeddelelse <br>**5:** Sender fejlen til admin                                                                                                             |
| Special Requirements                | **1:** Cross platform support<br>**2:** Håndterer edge cases hvor dobbelt booking kan forekomme                                                                                                                                                                                                                                                                         |
